# svn地址

# 基本架构

React18 +recoil + webpack + antd5

# 部分功能实现

  • 获取个人信息、判断是否登录等功能都是实现在APP.jsx文件中
  • 获取路由渲染此类的功能都是现在 components目录下的Routes组件中
  • 接口全部存放在API目录下，接口的请求地址配置到了.env文件中
  • 关于非组件的情况下调用弹窗和跳转,可以使用 eventBus(订阅发布模式)，部分功能已经在App.jsx注册，需要其他的功能可以继续在APP.jsx注册

# 钩子

  • 通过项目中的form钩子可以快速实现一套表格和对应的增删改查
  • useOnMountUnsafe钩子可以解决在开发环境useEffect钩子执行多次的问题
