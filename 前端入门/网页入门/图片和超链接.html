# 图片

标签定义 HTML 页面中的图像，有两个必需的属性：src 和 alt

<img src="smiley-2.gif" alt="Smiley face" width="42" height="42">
1

# 属性

属性 值 描述
src URL 规定显示图像的 URL
width 数字 规定图像的宽度
height 数字 规定图像的高度
alt string 规定图像的替代文本(图片不显示的时候显示alt中的文字)

# 超链接

<a href="https://www.taobao.com">一个购物网站</a>
1

# 属性

属性 值 描述
href URL 跳转的目标url
target _blank
_parent
_self
_top		 _blank：新窗口打开。
_parent：在父窗口中打开链接。
self：默认，当前页面跳转。
top：在当前窗体打开链接，并替换当前的整个窗体(框架页)。
