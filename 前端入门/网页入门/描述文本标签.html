# 描述一段文本

# small

用于描述一段声明类的文本html，所展现的样式就是可以使显示在浏览器中的文本变为较小号的字体

<p>w3cschool.cn - the world's largest web development site.</p>
<p><small>Copyright 1999-2050 by Refsnes Data</small></p>
1
2

# time

用于存放一段时间

<time>2023-2-22</time>
1

# abbr

用于存放简写的内容

<abbr title='是一个优秀的语言'>javascript</abbr>
1

# sub

下标

h<sub>2</sub>o
1

# sup

上标

5<sub>2</sub>
1

# del

标签定义文档中已删除的文本

<del>200元</del>
1

# ins

代表新插入部分的文本，一般来说 <del> 和 <ins> 一起使用，描述文档中的更新和修正。浏览器通常会在已删除文本上添加一条删除线，在新插入文本下添加一条下划线。

<p>My favorite color is <del>blue</del> <ins>red</ins>!</p>
1

# s

标记不再正确的文本

这个和 del标签有点像

<p><s>My car is blue.</s></p>
<p>My new car is silver.</p>
1
2

# code

标签是一个短语标签，用来定义计算机代码文本

<code>
    <?php
        echo 123;
    ?>
</code>

# progress

标签定义运行中的任务进度(不能实现实时进度)

<progress value="20" max="100">完成20%</progress>
1

中间的 完成20%默认看不到，浏览器不显示才会出现

# 强调文本

# strong

strong 标签和 em 一样，用于强调文本，但是它们的强调程度不同

<strong>我很重要</strong>
1

# em

倾斜标签，突出文本

<strong>我倾斜了</strong>
1

# mark

和mark笔一样，突出显示部分文本

<p>Do not forget to buy <mark>milk</mark> today.</p>
1

# 引用标签

# cite

标签定义作品（比如书籍、歌曲、电影、电视节目、绘画、雕塑等等）的标题。

<p><cite>The Scream</cite> by Edward Munch. Painted in 1893.</p>
1

# blockquote

用来定义摘自另一个源的块引用

下面来自名侦探柯南的引用
<blockquote cite="http://www.worldwildlife.org/who/index.html">
真相只有一个
</blockquote>
