flex-direction ​

flex-direction 用于控制主轴的方向

css
.container {
    flex-direction: row | row-reverse | column | column-reverse;
}
值描述
row主轴为水平方向，起点在左端
row-reverse主轴为水平方向，起点在右端
column主轴为垂直方向，起点在上沿
column-reverse主轴为垂直方向，起点在下沿
